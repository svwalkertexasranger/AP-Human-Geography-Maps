# AP Human Geography Maps

Interactive classroom map app for AP Human Geography.

## What Is Included

- World base map using Natural Earth country boundaries
- Continents map and quiz
- AP Human Geography regions map and quiz
- World religion map and quiz
- US base map
- US belts map and quiz
- US religion map and quiz
- Types of maps tiles and quiz
- Confetti/applause for correct answers
- Sad emoji/boo sound for incorrect answers
- Score, attempts, timer, and 3-wrong-answer shuffle behavior

## How To Run

Open `index.html` in a browser.

No server or install step is required. The app uses local files in:

- `assets/`
- `data/`
- `vendor/`

## GitHub Pages

This project can be hosted with GitHub Pages.

1. Upload all files and folders in this project to a GitHub repository.
2. In GitHub, go to `Settings` > `Pages`.
3. Set the source to the main branch and root folder.
4. Open the published GitHub Pages URL.

## Notes

The world map uses Natural Earth public-domain map data packaged locally through `data/atlases.js`.

The US Religion Map includes a classroom note about Texas: Catholics are a very large group there, but Pew's 2023-24 data lists Texas statewide at about 27% Evangelical Protestant and 22% Catholic.
